1. установка библиотеки ncurses:
sudo apt-get install libncurses-dev

2. подключить добавить заголовочный файл библиотеки в программу
#include <ncurses.h>

3. скомпилировать программу с указанием используемой библиотеки ncurses и подключением информации для отладки
gcc -g task9_part2_final_v3.c -o task9_part2_final_v3 -lncurses

только благодаря gdb смог наконец отыскать одну из последних причин непредсказуемого поведения программы

Описание:

1. каталоги помечаются жирным шрифтом
2. переход между страницами осуществляется с помощью клавиш Page UP и Page DOWN
3. переход между половинами (левая/правая) осуществляется клавишей TAB
4. при выходе из корневого каталога (для  юзера: home) возникает ошибка и программа закрывается
5. при длинных путях к файлу каталога портится рамка, не смотря на то, что текст переносится на следующую строку,
   похоже особенность используемой функции
6. в программе практически полностью отсутствуют проверки успешного выполнения функций связанных с каталогами: открытие, чтение, получение информации о файле и т.д.
7. выход из программы не предусмотрен, завершается принудительно ctrl+c
